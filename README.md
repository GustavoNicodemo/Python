# Estudos de Python

Este repositório contém exercícios e scripts desenvolvidos durante o aprendizado da linguagem Python, organizados por módulos temáticos.

## 🚀 Visão Geral

O projeto é uma coleção de scripts educacionais que abrangem desde operações básicas até lógica um pouco mais avançada, como cálculos de médias e manipulação de listas.

## 🛠️ Stack e Tecnologias

- **Linguagem:** Python 3.x
- **Frameworks:** Nenhum (uso de bibliotecas padrão)
- **Gerenciador de Pacotes:** N/A (não há dependências externas no momento)
- **IDE Sugerida:** PyCharm

## ⚙️ Configuração e Instalação

### Requisitos
- Python 3.6 ou superior instalado.

### Instalação
Não é necessária a instalação de bibliotecas externas. Basta clonar o repositório ou baixar os arquivos.

```bash
git clone <url-do-repositorio>
cd Python
```

## 🏃 Como Executar

Para executar qualquer script, utilize o interpretador Python diretamente no terminal a partir da pasta raiz:

```bash
python "03_listas\hello_junie.py"
```

Ou navegue até a pasta específica:

```bash
cd "02_operadores"
python media_harmonica.py
```

## 📝 Scripts Disponíveis

- **02_operadores/media_harmonica.py**: Script que calcula a média harmônica de notas com um fator de amortização.
- **03_listas/verificador_cpf.py**: Implementação de lógica para validação de dígitos de CPF.
- **05_estruturas_controle/exercicios_semana5_junie.py**: Exercícios de lógica condicional com refatoração para legibilidade e robustez.
- **06_estruturas_repeticao/exercicios_semana6_junie.py**: Prática de loops e estruturas de repetição com código otimizado.
- **07_vetores_matrizes/exercicios_semana7_junie.py**: Manipulação avançada de listas e matrizes com tratamento de exceções.
